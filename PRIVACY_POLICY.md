# Droplink Chrome Extension - Privacy Policy

**Last Updated: January 15, 2026**

## Introduction

Droplink ("we", "our", or "the extension") is committed to protecting your privacy. This Privacy Policy explains how our Chrome extension collects, uses, and protects your information.

## What We Collect

### Information You Provide
- **Email Address**: Used solely for authentication and account identification
- **Server Configuration**: Your self-hosted or configured Gotify server URL
- **Authentication Token**: Client token for connecting to your Gotify server

### Automatically Collected Information
- **None**: We do not collect any analytics, usage data, or tracking information

## How We Use Your Information

Your information is used exclusively for the extension's core functionality:

1. **Authentication**: Your email and token are used to authenticate with your Gotify server
2. **Message Receiving**: The extension connects to your configured server to receive link notifications
3. **Local Storage**: All settings and tokens are stored locally on your device using Chrome's storage API

## Data Storage

- **Local Storage Only**: All data is stored locally on your device
- **No Cloud Storage**: We do not store any of your data on our servers
- **User Control**: You can delete all stored data at any time by uninstalling the extension or clearing extension data

## Data Sharing

- **No Third-Party Sharing**: We do not share, sell, or transfer your data to any third parties
- **No Analytics Services**: We do not use any analytics or tracking services
- **Direct Connection**: The extension only communicates with your configured Gotify server

## Permissions Explanation

### storage
Used to save your settings, server configuration, and authentication token locally on your device.

### tabs
Used to open received links in new browser tabs, which is the core functionality of the extension.

### notifications
Used to show desktop notifications when new links are received from your mobile device.

## Security

- **HTTPS Only**: All connections use HTTPS encryption
- **Local Storage**: Sensitive data (tokens) are stored locally using Chrome's secure storage API
- **No Remote Code**: The extension does not load or execute any remote code
- **Open Source**: The extension code is open source and available for review on GitHub

## Your Rights

You have the right to:
- **Access**: View all data stored by the extension in the settings page
- **Delete**: Remove all stored data by uninstalling the extension
- **Control**: Enable or disable optional features (notifications, etc.)
- **Transparency**: Review the extension's source code on GitHub

## Children's Privacy

This extension is not intended for children under 13 years of age. We do not knowingly collect information from children.
